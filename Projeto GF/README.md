# Projeto Game Flow

## 1. Contexto
Este projeto é uma landing page focada em apresentar um protótipo ou conceito relacionado a jogos. Ele combina elementos visuais e multimídia para transmitir uma experiência de fluxo de jogo de forma clara e atrativa.

## 2. Público-alvo
O público-alvo são jovens e adultos interessados em jogos digitais, estudantes e entusiastas de design de experiência de usuário. O uso principal é em desktop e notebook, com suporte para visualização em telas maiores.

## 3. Dor principal
Atualmente, o usuário não consegue encontrar uma apresentação visual envolvente e organizada sobre o conceito do jogo de forma rápida e intuitiva.

## 4. Critério de sucesso
O usuário consegue entender o propósito do projeto e navegar pelas principais informações em menos de 30 segundos.
